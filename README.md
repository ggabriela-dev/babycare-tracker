# BabyCare Tracker 👶

Projeto simples para registrar atividades do bebê como mamadas, sono e troca de fraldas.

## Funcionalidades

- Registrar mamadas 🍼
- Registrar sono 😴
- Registrar troca de fralda 🧷
- Visualizar histórico das atividades

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Objetivo do projeto

Este projeto foi criado para praticar desenvolvimento web e lógica de programação.
